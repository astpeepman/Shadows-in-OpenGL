Были созданы новые сцены для демонстрации с учетом ошибок лабораторных работ. 
Управление осталось прежним:
клавиши 1-2 переключение сцен
WASD управление
left Shift - ускоренное управление
Колесо мыши - FOV

Имеется возможность изменять положения объекта в реальном времени через NumPad
Для этого необходимо в коде раскоментировать функцию moveModel и отправить туда нужную модель.

Из прошлых лабораторных работ были оставлены модели, освещение в шейдерах, материалы  с картами нормалей и т.д.
